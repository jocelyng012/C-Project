b. (5) 0x3FE00000
= 00111111 11100000 00000000 00000000
-> S = 0 (positive number)
-> E = 0111 1111 -> (2**6 + 2**5 + 2**4 + 2**3 + 2**2 + 2**1 + 2**0) = 127 - 127 = 0
-> M = 1100000 00000000 00000000 -> 0.1100000 00000000 00000000
-> 1 + (2**(-1) + 2**(-2)) = 1.75
= 1.75
